\# Security and Monitoring



\## Security Controls



\### AWS Network Firewall



AWS Network Firewall provides centralized inspection for:



\* Inter-VPC traffic.

\* Traffic between AWS and the on-premises environment.

\* Internet-bound traffic.



\### Site-to-Site VPN



\* Uses encrypted tunnels.

\* Uses IKEv2 for secure communication.

\* Uses BGP for dynamic route exchange.



\## Monitoring and Governance



\### AWS CloudTrail



CloudTrail records:



\* API calls.

\* Network configuration changes.

\* Administrative activities.



\### AWS Config



AWS Config provides:



\* Resource configuration tracking.

\* Compliance monitoring.

\* Change history and auditing.



\## Benefits



\* Improved security visibility.

\* Centralized monitoring.

\* Better compliance and auditing capabilities.

\* Enhanced governance across the AWS environment.
